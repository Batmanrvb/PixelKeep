# 🕹️ PIXEL KEEP — Retro Game Vault

A pixel art / 8-bit styled browser emulator for NES, SNES, Game Boy, and GBA.

## 🚀 Deploy to GitHub Pages

1. **Create a new GitHub repo** (e.g. `pixel-keep`)
2. **Upload these 3 files** to the repo root:
   - `index.html`
   - `style.css`
   - `app.js`
3. Go to **Settings → Pages → Branch: main → / (root) → Save**
4. Live at: `https://YOUR-USERNAME.github.io/pixel-keep/`

---

## 🎮 How to Play

1. Open the site
2. Drop a ROM file onto the upload area or click **[ SELECT FILE ]**
3. Click **▶ PLAY** on any game card
4. Your library is saved in your browser automatically

## ⌨️ Controls

| Key | NES/GB | SNES |
|-----|--------|------|
| Arrow Keys | D-Pad | D-Pad |
| Z | A | A |
| X | B | B |
| A | — | Y |
| S | — | X |
| Enter | Start | Start |
| Shift | Select | Select |

> Controls can be remapped inside the emulator's settings menu.

## 📁 Supported Formats

| System | File Extensions |
|--------|----------------|
| NES | `.nes` |
| SNES | `.sfc`, `.smc` |
| Game Boy / GBC | `.gb`, `.gbc` |
| Game Boy Advance | `.gba` |

## ⚠️ Legal

Only use ROM files for games you legally own. No ROMs are included.

## 🔧 Tech

- [EmulatorJS](https://emulatorjs.org) loaded from CDN — no backend needed!
- Pure HTML/CSS/JS — works on any static host
